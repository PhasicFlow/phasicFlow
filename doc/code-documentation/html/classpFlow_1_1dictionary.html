Detailed Description

Dictionary holds a set of data entries or sub-dictionaries that are enclosed in a curely braces or are in a file.

data entry format is:

entryName value;

and a dictionary in a file can be:

entryName2 value2;
entryName3 value3;
subDict1
{
entryName4 value4;
....
}
subDict2
{
...
}
